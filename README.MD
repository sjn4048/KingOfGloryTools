### 简介

关于王者荣耀的一些周边工具

### 目录介绍

- html-md2nga：一个用于将HTML/Markdown转化为NGA代码的工具。由于使用它搬运王者荣耀公告，因此放于此处。目前处于beta状态，成熟后移出单独建立仓库。
- toy：一些不成熟的小玩具
- 战令计算器：见[此处](<https://ngabbs.com/read.php?tid=16549128>)
